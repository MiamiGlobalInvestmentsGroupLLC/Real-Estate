# DealEdge AI – Real Estate Deal Intelligence Platform

DealEdge AI is a real-world SaaS platform designed to help real estate investors analyze, evaluate, and make faster decisions on investment opportunities.

The platform focuses on off-market deals, wholesaling opportunities, and buy-box matching, providing investors with instant insights and structured deal analysis.

---

## 🚀 Core Features

- Deal Analysis Engine (MAO, ROI, Profit Estimation)
- Buy / Negotiate / Pass Decision System
- Risk Level Assessment
- Investor-Oriented Deal Structuring
- Off-Market Opportunity Evaluation

---

## 🧠 What It Solves

Real estate investors often waste time manually analyzing deals and filtering bad opportunities.

DealEdge AI automates this process by:
- Structuring deal inputs
- Running instant calculations
- Highlighting profitable opportunities
- Reducing decision time from hours to seconds

---

## ⚙️ Tech Stack

- Frontend: Modern Web App (React / Next.js)
- Backend: Supabase / Serverless APIs
- Deployment: Vercel
